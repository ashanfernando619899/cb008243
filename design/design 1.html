Types of softeare engineers

There are two main types of software engineers

Applications Software Developers
    1. Applications software developers design computer applications,
      such as word processors and games, for consumers. They may create
      custom software for a specific customer or commercial software to
      be sold to the general public. Some applications software developers
      create complex databases for organizations.
Systems Software Developers
    1. A systems developer is a type of software developer. They develop and
      implement applications and programs for the backend processing systems
      used in businesses and organisations. They also test and modify systems
      to ensure that they operate reliably.
